# Music files for the playlist

Place your MP3 files here. The app expects these filenames (see `src/data/songs.js`):

- `perfect_ed_sheeran.mp3`
- `lover_taylor_swift.mp3`
- `die_with_a_smile_bruno_mars_and_lady_gaga.mp3`
- `at_last_etta_james.mp3`
- `cant_help_falling_inlove_with_u_elvis_presly.mp3`
- `thinking_out_loud_ed_sheeran.mp3`
- `all_of_me.mp3`

Rename your files to match, or update the `src` values in `src/data/songs.js`.
